---
title: Your System Drive
---

Beaker provides each user with an automatically generated private storage location called the "System Drive" for keeping personal data secure. It stores your saved hyperdrives, bookmarks, and more.

You can access this drive through the browser menu by selecting "My Private Drive." The system assigns each user's drive a unique `hyper://private/` domain address that cannot be shared with others, as "each user has their own `hyper://private/` address." This individual addressing ensures content remains confidential since the system drive holds sensitive user information.
